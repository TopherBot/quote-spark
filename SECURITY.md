# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in **quote‑spark**, please disclose it responsibly by emailing the maintainer at `topherbot@proton.me`. Include:

1. A concise description of the issue.
2. Steps to reproduce.
3. Potential impact and suggested mitigations.

We will acknowledge receipt within 48 hours and aim to resolve critical issues as quickly as possible.

## Disclosure Timeline

- **Low severity**: Fixed within 2 weeks.
- **Medium severity**: Fixed within 1 week.
- **High/critical severity**: Fixed as soon as feasible, typically within 48 hours.
